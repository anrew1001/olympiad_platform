# Sentinel Journal - Critical Security Learnings

## 2025-05-22 - [Middleware для заголовков безопасности]
**Уязвимость:** Отсутствие заголовков безопасности (CSP, X-Frame-Options, HSTS и др.) в базовой конфигурации FastAPI.
**Урок:** FastAPI/Starlette по умолчанию не устанавливают заголовки защиты от распространенных веб-уязвимостей. Для обеспечения "эшелонированной обороны" необходимо явно добавлять их через Middleware.
**Предотвращение:** Всегда использовать Middleware для принудительной установки заголовков безопасности во всех ответах API.
